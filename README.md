# Pokestir Website

Static website for Pokestir, prepared for free hosting with GitHub Pages.

## GitHub Pages

Expected Pages URL:

https://pokestirvgm.github.io/Pokestir-Website/

Configure GitHub Pages from the repository settings:

1. Open **Settings** > **Pages**.
2. Set the source to **Deploy from a branch**.
3. Select the `main` branch and the `/root` folder.
4. Save the settings and wait for GitHub to publish the site.

## Pages

- `index.html` - Home
- `gearAndSoftware.html` - Gear and libraries
- `links.html` - Social and music links
- `usage_terms_claims.html` - Usage terms and claims
